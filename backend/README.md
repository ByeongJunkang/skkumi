# SKKUMI - Backend


## 스프링 빌드
* cd backend
* ./gradlew build
* cd build
* cd libs
*  java -jar backend-0.0.1-SNAPSHOT.jar

## 로컬 DB 설정
* src -> main -> resources -> application.yml
* MySql db에서 skkumi 스키마 생성 혹은
* url: jdbc:mysql://localhost:3306/***, 로컬 db에 있는 스키마 이름을 ***에 넣어주기
* 

## Getting Started
로컬 머신 설정
* JDK 17+
* Mysql
